# US Stock Monitor

This repository keeps versioned snapshots of the US stock monitor project.

## Structure

- `v1.0/`: version 1.0 snapshot
- `v1.1/`: version 1.1 snapshot
- `v1.2/`: version 1.2 snapshot
- `v1.3/`: version 1.3 snapshot
- `v1.4/`: version 1.4 snapshot

Each version folder should be self-contained, including its own source files, watchlist, reports, tests, and version-specific notes when needed.
